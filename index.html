Client-side PDF tool

PDF 페이지 편집기

PDF를 서버에 업로드하지 않고 브라우저에서 페이지 삭제, 페이지 개수 기준 분할, ZIP 다운로드를 처리합니다.

1. PDF 불러오기

파일을 선택하면 페이지 미리보기를 생성합니다. 처리 과정은 현재 브라우저 안에서만 진행됩니다.

아직 선택된 PDF가 없습니다.

2. 작업 선택

삭제할 페이지를 체크하거나, 분할 기준 페이지 수를 입력하세요.

페이지 삭제 후 새 PDF 다운로드

아래 목록에서 삭제할 페이지를 체크한 뒤 실행하세요.

페이지 개수 기준 분할 ZIP 다운로드

예: 60페이지 PDF에서 10을 입력하면 1-10, 11-20처럼 나누어 ZIP으로 다운로드합니다.

3. 페이지 목록

PDF를 불러오면 페이지가 여기에 표시됩니다.
