Client-side document tool

PDF/PPTX 페이지 편집기

파일을 서버에 업로드하지 않고 브라우저에서 PDF 페이지와 PPTX 슬라이드 삭제, 개수 기준 분할, ZIP 다운로드를 처리합니다.

1. 파일 불러오기

PDF 또는 PPTX 파일을 선택하세요. PDF는 페이지 썸네일을, PPTX는 슬라이드 번호와 텍스트 요약을 표시합니다.

아직 선택된 파일이 없습니다.

2. 작업 선택

삭제할 페이지/슬라이드를 체크하거나, 분할 기준 개수를 입력하세요.

선택 항목 삭제 후 새 파일 다운로드

아래 목록에서 삭제할 항목을 체크한 뒤 실행하세요. 각 항목의 휴지통 버튼으로 즉시 목록에서 제거할 수도 있습니다.

개수 기준 분할 ZIP 다운로드

예: 60페이지 PDF에서 10을 입력하면 1-10, 11-20처럼 나누어 ZIP으로 다운로드합니다. PPTX도 같은 방식으로 슬라이드를 나눕니다.

3. 페이지 목록

파일을 불러오면 페이지 또는 슬라이드가 여기에 표시됩니다.
